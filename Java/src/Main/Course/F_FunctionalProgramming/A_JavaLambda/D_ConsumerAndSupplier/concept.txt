Predefined Functional Innterface:
    - Predicate -> test();
    - Function  -> apply(); 
    - Comsumer  -> accept();
    - Supplier  -> get();

1. Predicate adalah sebuah interface class yang memiliki satu method 
yaitu test() dengan memiliki satu argument yang dapay mengembalikan nilai 
boolean.

2. Function Interface juga memiliki satu method yaitu apply(). 
ini berfungsi untuk mengembalikan nilai yang dimasukkan / mengembalikan 
satu nilai dengan tipe data apapun (Object / Generic).

Bagaimana cara membuatnya? 
public R apply(T) {
    body...
}

Keterangan: 
    - T: tipe data generic
    - R: tipe data return

3. Consumer artinya akan menggunakan konsumsi, jadi bisa menerima sebuah argument atau input dan hanya menampilkan saja, tidak sampai  mengembalikan nilai inputannya. jadi hanya menerima inputan dan menampilkannya akan tetapi tidak mengembalikan inputan tersebut. hanya satu parameter dan tidak mengembalikan nilai apapun, hanya melakukan sebuah aksi saja.

4. Supplier artinya tidak membutuhkan sebuah parameter akan tetapi akan mensuplai sebuah nilai atau object dan mengembalikan nilai tersebut.

Atau lebih simpelnya:
- Predicate, seperti method isEmpty(), contains(), isBlank() (intinya hanya return boolean).
- Function, seperti method yang memiliki tipe data dan harus ada nilai yang akan direturn.
- Consumer, seperti method setter.
- Supplier, seperti method getter.
