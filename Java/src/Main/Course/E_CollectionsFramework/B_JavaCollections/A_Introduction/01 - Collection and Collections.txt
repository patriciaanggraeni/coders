Introduction to Java Collections

Permasalahan yang muncul pada Array:
    - Array menyimpan hanya dengan tipe data yang sama saja (homogeneus data type)
    - Array memiliki index yang terbatas (fixed size)
    - Array tidak mengalami peningkatan secara otomatis (tidak bisa ditambah dan dikurang saat program berjalan)
    - data stucture (tidak mempunyai method yang disediakan), misal ingin mengurutkan elemen array, maka fungsinya harus dibuat sendiri dan perlu logika untuk membuatnya

untuk mengatasi itu, java telah menyediakan collections framework (sudah disediakan banyak class di module java ini). jika tipe data Arraynya adalah object, maka bisa menyimpan tipe data yang beda beda (inilah definisi awal collections)

Collections vs Array (Perbandingan)
Pertama:
    - Array itu ukurannya fix
    - Collection itu ukuranya tidak fix (growable in nature)

Kedua:
    - Array hanya bisa menampung tipe data yang sama (homogeneus data)
    - Collections bisa menampung banyak tipe data sekaligus (heterogeneus data)

Ketiga:
    - Array tidak memiliki implementasi fungsi bawaan (jadi harus bikin manual fungsinya)
    - Collection mengimplementasikan data struktur (sudah ada fungsi bawaan)
